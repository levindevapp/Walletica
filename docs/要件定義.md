# 支出分析アプリ 要件定義書（Ver.0.1）

## 1. 概要

### 目的

日々の支出を細かく記録し、そのデータを様々な切り口で分析できるWebアプリケーション。

一般的な家計簿アプリのように「記録すること」が目的ではなく、

* 支出傾向の把握
* 支出割合の分析
* 前月・前年との比較
* 無駄遣いの可視化
* 固定費・変動費の把握
* 支出改善につながる情報の提供

を目的とする。

---

## 2. 基本コンセプト

「入力した支出データを、あらゆる切り口から分析できるアプリ」

利用者が知りたい情報をすぐ確認できるよう、多数のグラフ・ランキング・比較表示を提供する。
